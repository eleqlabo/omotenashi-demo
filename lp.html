FAQ

無料プランはいつまで使えますか？
無料プラン（Lite）はご利用期限なしでお使いいただけます。機能制限はありますが、AIによる文章生成・予約管理の基本機能を永続的にご利用いただけます。
LINEの実際の送信には何が必要ですか？
LINE公式アカウントの開設と、LINE Developersコンソールから取得した「Channel Secret」「Channel Access Token」が必要です。Standard プラン以上でLINE送信機能が有効になります。設定方法はサポートドキュメントでご案内しています。
既存の予約管理システムと連携できますか？
Pro・Platinumプランでは REST API を通じた外部連携に対応しています。お使いのPMSやOTAとの連携については、個別にご相談ください。
データのセキュリティは大丈夫ですか？
テナント単位でデータを完全分離しており、他施設のデータへのアクセスは技術的に不可能な設計です。通信はすべてTLS暗号化。国内データセンターを使用しています。
補助金・助成金の対象になりますか？
IT導入補助金（デジタル化基盤導入類型）の対象ツールとして申請中です。詳細は担当者にお問い合わせください。補助金申請のサポートも承っています。
解約はいつでもできますか？
はい、いつでも解約可能です。契約期間の縛りはありません（年払いの場合は残月分の返金はございません）。データは解約後30日間保持され、エクスポートが可能です。
